# Phase 4b — User Stories

**Goal:** Write INVEST User Stories with Given-When-Then acceptance criteria.

---

## Step 4.4 — User Story Writing

For each approved Use Case, write INVEST User Stories:

**Q44:** For [UC Name], how many User Stories are needed? (Break into independent, negotiable, valuable, estimable, small, testable stories)

---

## Step 4.5 — Acceptance Criteria

For each User Story, write Given-When-Then:

**Q45:** For [US Name]:
- Given [precondition]
- When [action]
- Then [expected result]

---

## Output

- `backlog/US-*.md` — individual user story files
- `story-inventory.md` — master list of all stories with INVEST status
