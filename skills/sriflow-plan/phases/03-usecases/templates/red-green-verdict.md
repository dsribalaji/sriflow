# UC Verdict Scoring Template

## Scoring Criteria

| # | Criterion | GREEN | RED |
|---|-----------|-------|-----|
| 1 | Goal level correct? | Sea Level or justified | Wrong level |
| 2 | Basic flow complete? | All steps, numbered, actor+system clear | Missing steps |
| 3 | Alternate flows? | At least 2 OR justified absence | None |
| 4 | Exception flows? | Trigger + system behavior + error message | Named but not specified |
| 5 | Preconditions? | Specific, testable | "System is running" |
| 6 | Postconditions? | Observable success AND failure | Missing |
| 7 | Business rules explicit? | Named, numbered, sourced | Embedded without declaration |
| 8 | No open questions? | All Q-# resolved or on a plan | Unresolved, no owner |
| 9 | Traces to BRD? | Linked to requirement | No traceability |

---

## Verdict

**GREEN** = all checks pass
**RED** = any check fails
