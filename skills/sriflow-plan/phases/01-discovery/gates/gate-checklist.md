# Phase 1 Gate — Discovery

## Tier-Specific Criteria

| Tier | Gate Criteria | Auto-Pass |
|------|---------------|-----------|
| **Small** | Problem clear + user identified + 3 acceptance criteria defined | Auto-pass (Phase 1 skipped) |
| **Medium** | Top 3-5 stakeholders named + top uncertainty identified | Manual check |
| **Enterprise** | Every Tier 1 stakeholder named with top uncertainty | Manual check |

**If tier = Small:** This phase was skipped. Auto-pass. Proceed to Phase 2.

| # | Criterion | Status | Evidence |
|---|-----------|--------|----------|

## Verification Checklist

- [ ] Stakeholder Register complete
- [ ] Power/Interest Map filled
- [ ] Uncertainty Tiers assigned (1/2/3)
- [ ] Disagreement Log reviewed
- [ ] Interview Plan confirmed

---

## Gate Decision

**PASS:** All criteria met → Proceed to Phase 2 (Elicitation)

**FAIL:** Identify gaps and ask more questions until criteria met
