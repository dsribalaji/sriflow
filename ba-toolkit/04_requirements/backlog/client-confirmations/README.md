# client-confirmations/

This folder stores SME and client sign-off confirmations for approved User Stories.

**File naming:** `CONFIRM-US-[###]_[YYYY-MM-DD]_[StakeholderName].md`

**What goes here:**
- Email threads (pasted as text) where a stakeholder confirmed an AC or business rule
- Meeting notes where a client confirmed scope
- Formal sign-off documents
- Screenshots of approval communications

**Why this matters:** When scope creep occurs or a stakeholder disputes a requirement in UAT,
these files are the evidence trail that confirms what was agreed and when.

**Update the story file:** When a confirmation is filed here, add it to the story's
`SME Confirmations` table in `backlog/approved/US-[###].md`.
